# AI Expense Tracker & Forecasting System

An AI-powered expense analysis tool that helps users track spending, understand financial habits, and predict future expenses using machine learning.

## Features

* Automatically categorizes expenses into Food, Travel, Health, Entertainment, Utilities, Education, Rent, Subscriptions, and Investments.
* Forecasts future spending for the next 1, 3, and 6 months using XGBoost and Prophet.
* Analyzes monthly and yearly spending trends.
* Provides category-wise expense breakdown and spending insights.
* Detects budget overspending and generates alerts.
* Identifies seasonal spending patterns and recurring expense trends.
* Supports CSV uploads for automated data analysis and forecasting.
* Displays results through interactive dashboards and visualizations.

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-Learn
* XGBoost
* Prophet
* Matplotlib
* Streamlit

## Workflow

1. Upload expense data.
2. Clean and preprocess transactions.
3. Categorize expenses using NLP.
4. Generate spending insights and visualizations.
5. Forecast future expenses using machine learning models.
6. Monitor budgets and spending trends.
